# Documento de Teste Imparcial para Sistema de Emissão Criptográfica

## Contexto
Este documento contém um algoritmo de emissão para uma criptomoeda que requer validação independente. O objetivo é verificar se o comportamento matemático está correto conforme as especificações.

## Algoritmo a ser Testado

```cpp
CAmount GetBlockSubsidy(int nHeight, const Consensus::Params& consensusParams)
{
    int halvings = (nHeight / 210000) % 33;
    if (halvings >= 64)
        return 0;
    CAmount nSubsidy = 50 * COIN;
    nSubsidy >>= halvings;
    return (nHeight < 70 * 210000 * 33) ? nSubsidy : 0;
}
```

## Especificações do Sistema

1. **Intervalo de Halving**: A cada 210,000 blocos
2. **Recompensa Inicial**: 50 unidades por bloco
3. **Mecanismo de Halving**: Divisão por 2 a cada intervalo
4. **Sistema de Ciclos**: 
   - Cada ciclo contém 33 halvings
   - Após 33 halvings, o sistema deve resetar para a recompensa inicial
   - Total de 70 ciclos permitidos
5. **Fim da Emissão**: Após completar 70 ciclos completos

## Constantes
- `COIN = 100,000,000` (unidade base)
- `nHeight` = altura do bloco (começando em 0)

## Questões para Análise

5. Qual seria a duração aproximada em anos (assumindo 4 anos por halving)?
